FeedFlow Trending

Hosts the weekly-curated "Trending This Week" feed list shown in the FeedFlow Android app's Discover screen.

Current manifest: v1/trending.json

How to update

Edit v1/trending.json in the repo, push to main. The app refreshes its cache once per 7 days via WorkManager and uses a bundled fallback list when this URL is unreachable.

Schema

{
  "version": 1,
  "updatedAt": "YYYY-MM-DD",
  "entries": [
    {
      "name": "Feed Name",
      "url": "https://example.com/feed.xml",
      "category": "Tech",
      "iconUrl": "https://www.google.com/s2/favicons?domain=example.com&sz=64",
      "description": "Short tag line",
      "rank": 1
    }
  ]
}
